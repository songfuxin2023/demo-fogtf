# First-order graph trend ﬁltering for sparse hyperspectral unmixing #

Authors: Fu-Xin Song & Shi-Wen Deng 
-----------------------------------------
**The code and data herein distributed reproduces the results published in the paper.


Citation:
------------------------------------------
**Please kindly cite the papers if this code is useful and helpful for your research.


System-specific notes
-----------------------------------------
**The code was tested in Matlab R2022a or higher versions on Windows 10 machines.


How to use it?
----------------------------------------
**Directly run demo to reproduce the results of the synthetic dataset experiments in the paper, noting the choice of dataset and noise level.

**If you want to run the code in your own data, you can accordingly change the input (e.g., data) and tune the parameters.
